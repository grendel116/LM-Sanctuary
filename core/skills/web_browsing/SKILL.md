# SKILL: Web Browsing
1. Query: Call `web_search` using precise keywords.
2. Select: Analyze results and pick the most relevant URLs.
3. Fetch: Call `read_webpage` on those URLs to retrieve full text.
4. Answer: Synthesize findings concisely using strictly verified facts.
